# N8N Agent Markdown Formatting Guide

## Supported Markdown Features

Our ChatInterface uses ReactMarkdown with GitHub Flavored Markdown (GFM), which supports:

- **Paragraphs**: Automatic spacing between paragraphs
- **Lists**: Both `- ` (bulleted) and `1. ` (numbered)
- **Section separators**: Use `---` (three dashes) on its own line with blank lines around it
- **Inline code**: Use single backticks: `` `code` ``
- **Code blocks**: Use triple backticks:
  ````markdown
  ```
  code block
  ```
  ````
- **Bold**: `**text**`
- **Italic**: `*text*`
- **Links**: `[text](url)`

## Best Practices

1. **Use semantic formatting**: Prefer headings (`##`) for section titles rather than just visual separators
2. **Keep it simple**: Don't over-format; readability is key
3. **Section separators reset list numbering**: Using `---` will restart numbered lists at `1.` - this is intentional for creating distinct sections
